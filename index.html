You cannot see it!

問題文

「~question/you_cannot_see_it!/see_it.txt」というフラグが書かれたファイルを見てみよう。しかしこのファイルは書き込みも読み込みも実行もできない。どうしたらフラグを入手することができるだろうか？

みてみよう

user01@raspberrypi:~/question_files/you_cannot_see_it $ cat see_it.txt
cat: see_it.txt: Permission denied 
↑ 権限がないといわれている！

読み込み権限の付与を行う

user01@raspberrypi:~/question_files/you_cannot_see_it $ chmod u+r see_it.txt
user01@raspberrypi:~/question_files/you_cannot_see_it $ cat see_it.txt
CST{There_was_no_authority}
user01@raspberrypi:~/question_files/you_cannot_see_it $

Challenge Crypto!

問題文

CTFにはCryptoという暗号分野に関する問題がある。簡単な暗号化には文字コードによる変換がよく用いられる。ファイル「~/question_files/Challenge_Crypto/encoded_flag」はフラグをROT13→Base64の順でエンコードしたものである。ファイルを復号してフラグを入手しよう。

エンコードを元に戻すにはデコードすればいい

Linuxの場合……

nkf, trコマンドでデコードできる 
user01@raspberrypi:~/question_files/Challenge_Crypto $ cat encoded_flag
UEZHe3BlbGNnYnRlbmN1fQ0K
user01@raspberrypi:~/question_files/Challenge_Crypto $ cat encoded_flag | base64 -d
PFG{pelcgbtencu}
user01@raspberrypi:~/question_files/Challenge_Crypto $ cat encoded_flag | base64 -d | nkf -r
CST{cryptograph}

from Crypto.Util.number import bytes_to_long as b2l

f = open("JPEG_FILE.jpg", "rb").read()

flag = open("flag.jpg", "wb")
